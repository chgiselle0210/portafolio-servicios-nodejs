# Portafolio de Servicios en Node.js

Proyecto que desarrollé para el curso **Backend con Node.js nivel avanzado** del bootcamp Fullstack Developer de Tecmilenio.

Construí la aplicación de forma gradual mediante distintos avances. Cuenta con un servidor con Express, demostración del Event Loop, arquitectura modular, middleware de seguridad, validación de datos, registro e inicio de sesión, autenticación con JWT y autorización por roles.

## Tecnologías utilizadas

- Node.js
- Express
- JSON Web Token
- bcryptjs
- express-validator
- Helmet
- express-rate-limit
- dotenv

## Requisitos

- Node.js instalado
- npm
- Postman para probar los endpoints
